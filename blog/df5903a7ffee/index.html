启动

  • win + r 输入 cmd

命令

  • cmd大小写不敏感
  • 输入文件或者文件名的首字母Tab键补全文件名
  • ↑ ↓ 查看历史命令
  • ESC 清除当前命令
  • CLS 清屏
  • mkdir 创建文件夹
  • exit 退出
  • /? 查看命令帮助 cd/?
  • cd 回到根目录
  • cd,, 返回上级目录
  • REM：命令后内容不执行但回显
  • :: 不回显注释
  • @控制该行命令是否回显
  • echo on off 控制是否回显
  • echo ^G鸣响
  • title 设置cmd窗口标题
  • color 1A 设置控制台前景和背景颜色
  • goto :
  • start firefox 启动
  • pause 暂停
  • if string1==string2 command
  • if exist filename command
  • >将运行结果传递到后面
  • echo hello > 1.txt 建立一个文件内容为“hello“
  • >>传递并末尾追加
  • command1|command2 第一条命令的结果作为第二条命令的参数
  • fortune | cowsay
  • ^ 转义字符 对<,>,& 进行转义，去除其特殊功能
  • “”允许文件名有空格(“fortune cowsay”)
  • ! 变量延迟问题中表示变量 !var!
